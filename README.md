# 💕 Romantic Website

A beautiful, interactive website filled with love, comfort, encouragement, and admiration messages for your special someone.

## Features

✨ **Interactive Buttons**
- Comfort Me: Soothing and supportive messages
- Encourage Me: Motivational and empowering messages
- Admire Me: Compliments and appreciation
- Love Declaration: Deep, romantic proclamations

🎨 **Visual Effects**
- Animated gradient background
- Floating text elements
- Heart beat animations
- Pop-up modals with smooth transitions
- Confetti and sparkle effects
- Responsive design for mobile and desktop

💌 **Messages**
- 5 different messages for each button
- Heartfelt and personalized content
- Randomly selected for variety

## How to Use

1. Open `index.html` in your web browser
2. Click any of the four buttons to see romantic messages
3. Each button reveals different types of messages
4. Close the pop-up and click again for different messages

## Customization

You can easily customize the messages by editing `script.js`:
- `comfortMessages[]` - Change comfort/support messages
- `encourageMessages[]` - Change motivational messages
- `admireMessages[]` - Change admiration messages
- `proposeMessages[]` - Change love declaration messages

Add more messages or personalize them with specific details!

## Deployment

You can deploy this website for free using:
- **GitHub Pages**: Enable in repository settings
- **Netlify**: Drag and drop the files
- **Vercel**: Connect your GitHub repository
- **Any web hosting service**

## Technologies Used

- HTML5
- CSS3 (Animations & Gradients)
- Vanilla JavaScript
